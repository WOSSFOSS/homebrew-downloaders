# WOSSFOSS Downloaders

## How do I install these formulae?

`brew install wossfoss/downloaders/<formula>`

Or `brew tap wossfoss/downloaders` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "wossfoss/downloaders"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
